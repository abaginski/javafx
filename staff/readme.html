JavaFX Nutshell example STAFF

This Nutshell example implements a third party shared library called "ControlsFX", which is an open source project that aims to provide additional high quality GUI controls to JavaFX. 

You can get the latest version of ControlsFX here: <http://fxexperience.com/controlsfx/>. STAFF uses ControlsFX in Version 8.40.12 and demonstrates the usage of the following control classes in BSF4ooRexx:

  • Notifications
  • BreadCrumbBardssd
  • TableRowExpanderColumn
  • ToggleSwitch
  • MaskerPane
  • Rating

This Application consists of a TableView with predefined data. You can change the data by clicking on items of the BreadCrumbBar located at the top right corner of the window. By activating the TableRowExpanderColumn marked with a "+" in the table a Rating bar shows up to rate each employee. Every user interaction in the STAFF nutshell example is followed either by the MaskerPane appearing or a Notification.

To start this Application, the JAR File located in the root directory has to be included in the environmental variable "CLASSPATH". Luckily, STAFF provides a way to do this on-the-fly using one of the following shell files:

  • [run.cmd] for Windows OS
  • [run.sh] for MacOS and Linux
